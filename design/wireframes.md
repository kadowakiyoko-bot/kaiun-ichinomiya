# 開運一の宮めぐり — ワイヤーフレーム v1.0

**対象ユーザー**: 40〜60代女性  
**実装スタック**: Next.js 14 + Tailwind CSS v4 + React Leaflet  
**デバイス優先順位**: Mobile (375px) → Tablet (768px) → Desktop (1024px)  
**作成日**: 2026-04-22

---

## 画面一覧

| No. | 画面名 | URL想定 | 主要コンポーネント |
|-----|--------|---------|------------------|
| 1 | ホーム（ダッシュボード） | `/` | ProgressGauge, NearbyShrines, RecommendPanel |
| 2 | 地図画面 | `/map` | ShrineMap, FilterPanel, ShrineCard |
| 3 | 神社詳細 | `/shrine/[id]` | ShrineDetail, StampButton, OmamoriList |
| 4 | マイページ | `/mypage` | StampBook, AchievementList, VisitHistory |
| 5 | 旧国別一覧 | `/region/[koku]` | RegionHeader, ShrineList |

**コンポーネント仕様**

ProgressGauge（参拝進捗）:
```
- 数字: Cormorant Garamond 36px / #BB3D20
- テキスト: "/ 102 社" / Noto Sans JP 14px / #7A6555
- バー: height 8px, border-radius 9999px
         background track: #E8E2DB
         fill: linear-gradient(to right, #BB3D20, #D4AF37)
- バーアニメ: mount時に0→実値へ 0.8s ease-out
```

NearbyShrineCard（近くの一の宮）:
```
- 位置情報取得前: 「位置情報を許可して近くの神社を見つけましょう」
                  [許可する] ボタン（Primary）
- 距離表示: "36km" / Cormorant Garamond 13px / #BB3D20
- 訪問済み判定: 左ボーダーを #D4AF37 に変更
```

BottomNavigation:
```
- 高さ: 64px（セーフエリア対応: padding-bottom: env(safe-area-inset-bottom)）
- タッチターゲット: 各タブ幅 25% × 64px = 十分な44px以上
- アクティブ色: #BB3D20
- 非アクティブ色: #B5A899
- フォント: Noto Sans JP 10px
```

## インタラクション設計メモ

### スタンプ押下フロー

```
1. ユーザー: 「スタンプを押す」ボタンタップ
   ↓
2. 確認ダイアログ（モバイル: ボトムシート）
   「鹿島神宮 (茨城県) を参拝記録しますか？」
   [参拝した！] [キャンセル]
   ↓
3. [参拝した！] タップ
   ↓
4. スタンプドロップアニメーション（0.4s）
   墨にじみエフェクト（0.6s）
   ゴールドパルス（1.0s）
   ↓
5. aria-live="assertive" で
   「No.001 鹿島神宮の参拝を記録しました」をスクリーンリーダーに通知
   ↓
6. 称号解除チェック → 解除時はさらにモーダル表示
   「称号解除: 初の一の宮 参拝おめでとうございます」
```

### オフライン対応メモ

```
神社データ: 初回ロード時に localStorage / IndexedDB にキャッシュ
訪問記録:   オフライン時もローカル保存 → オンライン復帰時に同期
地図タイル: react-leaflet の TileLayer キャッシュ設定（maxAge: 604800）
```

### 40〜60代女性向けモバイル配慮

```
- 文字サイズ: body最小14px、できれば15px推奨
- 行間: 1.7以上（読みやすさ優先）
- ご利益・難読語: 必ずルビ付与
- ボタン: ラベルを長めに（「見る」より「詳しく見る」）
- エラーメッセージ: 赤で目立たせ、解決策を必ず添える
- ローディング: "読み込み中..." を表示（スピナーのみにしない）
- 成功フィードバック: 「参拝記録しました！」と必ず文字で伝える
```
